# Contact Service C++ (VERSION 1.0)

This project is a C++ translation of my CS 320 Java Contact Service. This current version is more
or less the same, but translated from Java to C++. Some quality of life changes were made along with
the replacement of JUnit testing for Google test (compatibility).

## Features in this project

- Contact model that has validation rules
- ContactService with CRUD operations
- Automated Testing with googletest
- CMake introduction.
